# Move It - Hotel Booking Site

## Overview

Move It is a responsive hotel booking website template built using **HTML**, **CSS**, and **Bootstrap 5.3**. It is designed to showcase hotel services and accommodations with a clean, modern user interface. The project includes a responsive navigation bar, an image carousel, content cards, and a structured footer with contact and social media links.

## Features

- **Responsive Navbar**: Collapses into a hamburger menu on smaller screens.
- **Image Carousel**: Displays rotating hotel images.
- **Card Layout**: Sections displaying various hotel services or rooms.
- **Footer**: Includes contact information and social media links.
- **Responsive Design**: Ensures the site looks great on mobile, tablet, and desktop screens.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout design.
- **Bootstrap 5.3**: For responsiveness, grid layout, and components like the navbar, carousel, and cards.
